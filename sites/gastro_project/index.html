  • «‎Инструкция»‎

    Открывается окно с инструкцией по применению. Время для прочтения останавливается или дается дополнительно. Доступ к инструкции постоянный, при использовании не блокируется.

  • «‎50:50»‎

    После использования подсказки 50 на 50 игровой компьютер убирает два заведомо ложных варианта, оставляя игроку один верный ответ и один неверный.

  • «‎15 сек»‎

    Добаляет 15 секунд для ответа на вопрос.

  • «‎Совет эксперта»‎

    К каждому вопросу заранее готов комментарий медицинского эксперта с подсказкой верного варианта ответа.
